Why PDFast?

The PDF viewer built into Microsoft Edge stopped being good. After Edge switched its underlying PDF engine, everyday viewing got slower and buggier — and the setting that let you fall back to the old, reliable viewer was quietly removed.

PDFast brings that experience back: a fast, no-nonsense, native viewer that opens instantly, scrolls smoothly and renders crisply — plus the editing and document tools the built-in viewer never had.

Everything you need, nothing you don't

A complete PDF toolkit that stays small, quick and native.

Fast, crisp viewing

Smooth continuous scroll, zoom, Fit Width & Fit Page, page drop shadows, and sharp rendering on HiDPI displays — powered by MuPDF.

Tabs & multi-doc

Open several PDFs at once in tabs. Drag to reorder, middle-click or Ctrl+W to close. Opens maximized with native Windows 11 chrome.

Annotate & fill forms

Highlight, draw freehand and add text boxes — all edited inline, no modal dialogs. Fill AcroForm fields right on the page.

Find & print

Incremental Find (Ctrl+F) with a live match counter and jump-to-next. Print via the standard Windows dialog with page-range support.

Security

Open password-protected PDFs with an inline prompt, and remove passwords or restrictions from files you're able to open.

Document tools

Organize, merge, split, resize to A4, flatten to read-only, compress, and truly redact content — all from one wrench menu.

Automatic updates

Quietly checks GitHub for a newer release on launch and offers a one-click update — downloads, swaps itself in, and relaunches.

Truly standalone

One ~5.5 MB .exe. No installer, no .NET, no VC++ redistributable — only stock Windows DLLs. Copy it anywhere and run.
